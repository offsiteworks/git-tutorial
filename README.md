## 練習用テストリポジトリのクローン

練習用テストリポジトリのクローンからやりましょう。
ローカルPCにどこでもいいのでGitワークフォルダとして空のフォルダを作成してください。

`C:\work` でも結構です。


右クリック → Git クローン(複製)...<br/>
